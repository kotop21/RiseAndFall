# 🗺️ Установка кастомных карт

Данное руководство описывает процесс установки пользовательских карт в игру.

---

## 📥 1. Скачивание карты

Скачайте архив или набор файлов с картой от других игроков (обычно `.zip`, `.rar` и т.п.).

После скачивания распакуйте архив в удобное место.

---

## 📂 2. Структура файлов карты

Кастомные карты обычно содержат один или несколько типов файлов:

* `.scn` — сценарии (готовые карты для игры)
* `.rmv` — скрипты генерации карт

---

## 📁 3. Копирование файлов

Переместите файлы в соответствующие папки в директории игры:

**Сценарии (.scn):**

```
<DataPath>/Data/Scenarios/
```

**Скрипты карт (.rmv):**

```
<DataPath>/Data/Maps scripts/
```

> Где `<DataPath>` — это папка с игровыми ресурсами (может задаваться параметром запуска `-datapath`).

---

## ▶️ 4. Запуск игры

После копирования файлов:

1. Запустите игру
2. Перейдите в меню выбора карт
3. Найдите установленную карту в списке

---

## 💡 Советы

* Храните оригинальные файлы отдельно (на случай замены)
* Проверяйте источник скачивания карт
