# 🗺️ Custom Maps Installation

This guide explains how to install community-made maps into the game.

---

## 📥 1. Downloading a Map

Download an archive or a set of files containing a map created by other players (usually `.zip`, `.rar`, etc.).

After downloading, extract the archive to any convenient location.

---

## 📂 2. Map File Structure

Custom maps usually include one or more of the following file types:

* `.scn` — scenarios (ready-to-play maps)
* `.rmv` — map generation scripts

> ⚠️ **Important:** Both `.scn` and `.rmv` files are required for the map to work correctly. If only one of them is present, the map will not function.

---

## 📁 3. Copying Files

Move the files into the appropriate folders inside the game directory:

**Scenarios (.scn):**

```
<DataPath>/Data/Scenarios/
```

**Map scripts (.rmv):**

```
<DataPath>/Data/Maps scripts/
```

> `<DataPath>` is the folder containing game resources (can be set via the `-datapath` launch parameter).

---

## ▶️ 4. Launching the Game

After copying the files:

1. Launch the game
2. Open the map selection menu
3. Find the installed map in the list

---

## 💡 Tips

* Keep original files as backup (in case of replacement)
* Download maps only from trusted sources
